# Architecture Decision Records (ADR)

> Этот каталог содержит записи архитектурных решений (ADR) для AI Roleplay Coach Hub.

---

## Что такое ADR?

Architecture Decision Record — это короткий документ, фиксирующий значимое архитектурное решение, включая контекст, рассмотренные альтернативы и выбранный подход. ADR иммутабельны — после принятия они не редактируются; новый ADR заменяет старый.

## Процесс ADR

1. **Предложение** — Любой участник может предложить ADR
2. **Обзор** — Команда обсуждает и рецензирует
3. **Принятие** — ADR принят и добавлен в репозиторий
4. **Иммутабельность** — Принятые ADR никогда не изменяются (при необходимости заменяются новым ADR)
