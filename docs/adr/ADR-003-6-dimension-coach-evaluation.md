# ADR-003: 6-мерная оценка Coach

**Статус:** Принято

**Дата:** 2026-07-12

## Контекст

Coach Agent должен оценивать навыки общения пользователя. Оценка должна быть многомерной, объяснимой и полезной для обучения.

## Решение

Принять **6-мерную систему оценки** с sandwich-фидбеком:

### Шесть измерений
| Измерение | Вес | Что измеряет |
|-----------|-----|--------------|
| empathy | 20% | Понимание чувств клиента |
| listening | 20% | Перефразирование, подтверждение |
| problem_solving | 20% | Предложение решений, альтернатив |
| handling_aggression | 20% | Деэскалация, спокойствие |
| communication | 10% | Ясность, активное слушание |
| structure | 10% | Открытие, закрытие, логика |

### Оценка
- Каждое измерение: 0-100
- `overall_score = взвешенное_среднее(измерения)`
- Сильные стороны: измерения с `score >= 70`
- Слабые стороны: измерения с `score < 50`

### Формат фидбека
- **Sandwich:** Начать с сильной стороны → область для улучшения → закончить ободрением

## Рассмотренные альтернативы

- **Единая оценка:** Просто, но не даёт actionable информации — отклонено.
- **Бинарный pass/fail:** Слишком грубо — отклонено для учебного use case.
- **Free-text LLM фидбек:** Гибко, но нестабильно — rule-based оценка обеспечивает воспроизводимость.

## Последствия

- **Положительно:** Объяснимая, консистентная оценка
- **Положительно:** Чёткий путь обучения — пользователь знает, какое измерение улучшать
- **Положительно:** Детерминированно — одинаковый вход всегда даёт одинаковый выход
- **Отрицательно:** Лингвистические маркеры основаны на правилах — могут упускать нюансы общения
